## Why React Scanner Studio?

Understanding how components are used across a large codebase can be challenging. React Scanner Studio bridges the gap between raw scan data and actionable insights by providing:

- **Visual component usage statistics** â€” See which components are most used at a glance
- **Prop analysis** â€” Understand which props are commonly used and which are rarely touched
- **Shareable reports** â€” Build static dashboards that can be hosted on any static file server
- **Design system metrics** â€” Track adoption of your design system components over time

## Quick Example

```bash
# Initialize configuration (one-time setup)
npx react-scanner-studio init

# Start the interactive dashboard
npx react-scanner-studio start

# Build static files for sharing (optional)
npx react-scanner-studio build
```
